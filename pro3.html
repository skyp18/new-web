Arduino Simple Dice

The Arduino Simple Dice is a beginner-friendly embedded systems project that simulates a digital dice using LEDs and a push button. When the user presses the button, the LEDs display a random number from 1 to 6, mimicking the roll of a physical dice. Ideal for learning basic electronics and microcontroller programming.

Technologies Used

  • Arduino (Uno / Nano)
  • LEDs (7-segment or individual)
  • Push Button
  • Resistors & Breadboard
  • C/C++ (Arduino IDE)
